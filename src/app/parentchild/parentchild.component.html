You can think of Event Emitters as nothing but a means to bridge the gap or provide a means of communication between parent and child components. This is one of the first things you learn in Angular as everything in Angular is components and thats waht help modularize the code.
If you want to notify the child component of any change from the parent component make use of the Input() decorator.
If you want to notify the parent component about something from the child component make sure to use Output() decorator along with a Event Emitter.
